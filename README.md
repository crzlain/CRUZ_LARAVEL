# Lorraine Cruz Final Laravel Requirement

This project is a personal portfolio website created as part of the final requirement for the Web Development course. It showcases the developer’s background, skills, projects, and achievements in a clear and professional format. The portfolio also serves as an online representation of the developer’s professional profile and can be used for internship or employment opportunities.

# Features

Home Section – Displays a professional introduction with a brief overview of the developer.

About Section – Contains personal background information and a short description.

Skills Section – Lists technical and creative skills in an organized layout.

Projects Section – Highlights selected works with images and brief descriptions.

Contact Section – Provides a contact form for inquiries or collaboration.

Responsive Layout – Adapts automatically to different screen sizes for desktop, tablet, and mobile devices.

Theme -  Light and Dark mode of thes system

#Technologies Used

HTML5
CSS3
JavaScript

# System Requirements

Any modern web browser (Google Chrome, Mozilla Firefox, or Microsoft Edge)
Code editor (Visual Studio Code or any text editor)
